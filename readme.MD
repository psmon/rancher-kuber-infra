# 자신의 클러스터를 추가해보자

기본 클러스터 구성이 되어있으며 Add Cluster를 통해 추가 가능합니다.

모든 클라우드 쿠버네틱 리소스와 호환가능합니다.

클러스터 추가는 곧 클라우드 장비의 기부와도 같으니 학습용 장비쉐어의 뜻이 있으신분 추가하셔도되겠습니다.
